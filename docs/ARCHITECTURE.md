# 6IDE7 — Архитектура и концепция

## Обзор проекта

**6IDE7** — это инновационная интегрированная среда разработки (IDE), предназначенная для создания программных приложений методом визуального программирования.

### Ключевая концепция

Использование готовых блоков кода, которые пользователь может перетаскивать на рабочую область и соединять между собой для формирования логики приложения.

### Основные преимущества

- **Низкий порог входа** — не требует глубокого знания синтаксиса
- **Визуальное представление** — понятная структура программы
- **Мгновенный просмотр кода** — изучение реального синтаксиса
- **Высокая производительность** — реализация на Rust
- **Кроссплатформенность** — Windows, macOS, Linux

## Архитектура системы

### Слои архитектуры

| Слой | Компоненты | Ответственность |
|------|------------|-----------------|
| Presentation | UI Components, Event Handlers, Theme Engine | Визуализация интерфейса |
| Application | Editor Controller, Project Manager, State Machine | Координация компонентов |
| Domain | Block Models, Connection Graph, Code Templates | Бизнес-логика |
| Infrastructure | File System, Plugin Loader, Code Generator | Работа с ресурсами |

### Технологический стек

- **Язык**: Rust (edition 2021)
- **GUI**: egui + eframe
- **Сериализация**: serde
- **Асинхронность**: tokio
- **Парсинг**: tree-sitter

## Основные компоненты

### 1. Редактор визуальных блоков

Центральный компонент UI, обеспечивающий:
- Рабочую область (Canvas) для размещения блоков
- Панель блоков (Block Palette) с библиотекой компонентов
- Панель свойств (Properties Panel) для редактирования
- Drag-and-Drop механизм соединения блоков

### 2. Менеджер блоков кода

Отвечает за:
- Хранение и организацию библиотеки блоков
- Загрузку встроенных и пользовательских блоков
- Систему плагинов для расширяемости

### 3. Генератор кода

Преобразует визуальное представление в код:
- Анализ графа блоков → AST
- Оптимизация кода
- Генерация для Python, JavaScript, TypeScript, Rust, C++

### 4. Модуль интеграции с языками

Обеспечивает:
- Импорт существующего кода
- Интеграцию с пакетными менеджерами
- Запуск и отладку программ

```
├── Cargo.toml              # Workspace configuration
├── crates/
│   ├── ide-app/            # Main application
│   ├── ide-core/           # Core types and traits
│   ├── ide-blocks/         # Block manager
│   ├── ide-generator/      # Code generator
│   ├── ide-ui/             # UI components
│   └── ide-lsp/            # Language integration
├── plugins/                # Plugin examples
├── blocks/                 # Built-in block definitions
└── docs/                   # Documentation
```
